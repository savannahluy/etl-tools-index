Contributions are welcome! 

This repository accepts **commercial, open-core, and open-source tools**, as long as they are actively used in production environments.

### Inclusion Guidelines

Tools should:
- Solve a real ETL / ELT / data integration problem
- Be actively maintained and available today
- Have public documentation or product pages
- Fit one or more of the listed categories

### How to Contribute

1. Fork the repo  
2. Add a new tool using the standard format  
3. Open a Pull Request with a short description  

See **CONTRIBUTING.md** for the full template.
